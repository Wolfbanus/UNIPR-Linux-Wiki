## $\LaTeX$
- via APT
  - copiare, incollare (ctrl + shift + V) il seguente comando sul terminale e cliccare il tasto "invio". 
  Per trovare il terminale cercare "terminale" tra le app e aprirlo
  ```
  sudo apt install texlive-full
  ```
  vi chiederà conferma del download, digitare "S" e premere "invio"
  
  - installare il software TexStudio
 
- via Flatpak
  - installare TexStudio da Flathub
